Configuration Steps
	1) Create a environment variable "CATALOG_ENV"
	2) Create conf, logs & docs folder on the above location
	2) Copy config.properties and log4j.properties files in the "conf" folder 
	3) Copy given excel files in the "docs" folder 
	3) Update conf/config.properties 
	4) Update conf/log4j.properties 
	5) Copy "lib" folder and CatalogUtil.jar file from given Executable folder and place anywhere on server 
	
Utility execution steps
	6) java -jar CatalogUtil.jar servercatalog
